Evaluación de Calidad del Software

Analiza y mide la calidad de tus aplicaciones según criterios de ingeniería de software reconocidos internacionalmente.

Componentes Principales

Usabilidad

Evalúa la facilidad de uso e interacción del usuario con el sistema. Se consideran aspectos como accesibilidad, diseño y experiencia del usuario.

Fiabilidad

Mide la capacidad del software de mantener un desempeño estable bajo distintas condiciones y durante un tiempo prolongado.

Mantenibilidad

Determina la facilidad con que el software puede ser actualizado, corregido o mejorado sin comprometer su funcionalidad.

Método de Evaluación y Fórmula

Método de Evaluación

Se utiliza una escala del 1 al 5 para cada criterio, donde 1 representa deficiencia y 5 excelencia. Cada valor es ponderado para obtener un promedio general.

  • Recolección de datos mediante calificación individual.
  • Análisis comparativo entre criterios.
  • Visualización con gráficas dinámicas.
Fórmula General

El índice de calidad (IC) se obtiene con la siguiente expresión:

IC = (ΣCriterios) / N

Donde ΣCriterios es la suma de los puntajes de cada criterio evaluado y N es el número total de criterios considerados.
